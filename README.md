# Tebakkasuslove

## Project Description
Tebakkasuslove is a fun and engaging web application designed to stimulate your mind while showcasing your love for guessing games. Players can enjoy a range of interactive puzzles and challenges that test their knowledge and instinct.

## Setup Instructions
1. **Clone the Repository**  
   To get started, clone the repository to your local machine:
   ```bash
   git clone https://github.com/tebakkasus/Tebakkasuslove.git
   cd Tebakkasuslove
   ```
2. **Install Dependencies**  
   Use npm or yarn to install the required dependencies:
   ```bash
   npm install
   ```
3. **Start the Application**  
   Launch the application locally:
   ```bash
   npm start
   ```

## Tech Stack
- **Frontend:** React.js, Redux, CSS
- **Backend:** Node.js, Express
- **Database:** MongoDB

## Features
- Interactive guessing games that challenge your intellect.
- User-friendly interface for a seamless experience.
- Real-time score tracking for player performance.
- Community features allowing users to connect and share challenges.
